# NeuroFlow-FL 🚦🤖

**NeuroFlow-FL** is a Reinforcement Learning based traffic signal control project that aims to reduce traffic congestion by optimizing traffic light timing in a simulated road network.

The project uses **Deep Q-Network (DQN)** and **Proximal Policy Optimization (PPO)** to train traffic signal agents in the **SUMO** simulation environment. It also includes **Emergency Vehicle Preemption** and a planned **Federated Learning** architecture for privacy-preserving multi-intersection learning.

---

## Features

- Adaptive traffic light control using **Reinforcement Learning**
- Comparison of **DQN** and **PPO** algorithms
- **Single-intersection** and **multi-intersection** traffic simulation
- **Emergency vehicle preemption** support
- Designed for future extension with **Federated Learning**
- Built using open-source tools like **SUMO**, **Python**, **PyTorch**, and **Stable-Baselines3**

---

## Project Objectives

- Reduce average vehicle waiting time at traffic intersections
- Compare the performance of DQN and PPO for traffic signal control
- Extend control from one intersection to multiple intersections
- Support emergency vehicle priority without breaking learned policies
- Explore Federated Learning for decentralized training
